# Playable 解包工具

把从 InsightTracker 下载的 AppLovin 加载器（几 KB 的 HTML）还原成完整的 Playable 文件。

## 在线使用
👉 https://sven0920.github.io/playable-unpacker/

纯前端，打开网页上传加载器 HTML 即可解包，文件不上传服务器（AppLovin CDN 允许跨域）。

## 文件
- `index.html` — 浏览器版解包工具，由 GitHub Pages 托管

## 更新
改完 `index.html` 后 `git push`，GitHub Pages 几十秒后自动重新部署。
